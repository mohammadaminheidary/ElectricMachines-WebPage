# Electric Machines Web Page

A clean and simple front-end web page introducing electric vehicles (EVs), their benefits, types, and key technologies.

## Features
- Responsive design
- Introduction to different types of electric machines (BEV, PHEV, HEV)
- Basic styling with CSS
- Well-structured HTML

## Technologies Used
- HTML5
- CSS3
- (Optional: JavaScript if you add any)

## Purpose
This project was created as a practice / showcase for a web page focused on electric mobility and clean energy.

## How to Use
Just open `index.html` in any modern browser.
